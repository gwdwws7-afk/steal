## 第一部分：世界观完整设定

---

### 1.1 一句话概念

> 在被淹没的低地之后，安第斯高原成了人类最后的庇护所。Inti Imperium——太阳帝国——在这里用神经科技重建了印加神权。一个精英军官发现了印加预言的真相：她的血统不是拯救帝国的钥匙，而是毁灭的导火索。

---

### 1.2 世界背景

**时间线**
- **2089**：海平面上升，低地文明崩溃
- **2095-2150**：安第斯高原难民涌入，建立城邦
- **2180**：十二大家族联合建立 Inti Imperium，神权科技寡头
- **2247**：当前时间，帝国第 67 年统治

**地理**
- 核心区域：高原核心都市"库斯科·新日"（Nova Cusco）
- 五大区域对应游戏五关：工业区、议会塔、研究所、Qhipu 核心、Inti 太阳殿
- 周边：安第斯高原地形，气候极端，有"禁区"（帝国不愿公开的区域）

---

```
Inti Imperium 太阳帝国
│

├── Inca Tech（军事-工程复合体）

### 1.5 核心哲学主题

**主线命题**
> 预言说太阳神的归来意味着帝国的毁灭——但帝国将"毁灭"解读为"净化"，认为毁灭的是旧秩序，建立的是永恒神权。**预言是诅咒还是解放？**

**三个对立轴**
| 轴 | 一端 | 另一端 |
|---|---|---|
| **时间观** | 印加循环时间（太阳历/农业周期） | 赛博线性进步（数据永生） |
| **血统论** | 血统决定命运（神权合理化） | 血脉是枷锁（反叛逻辑） |
| **集体/个体** | 太阳祭的集体神经同步（统一意志） | 自由意志（玩家的核心诉求） |

### 1.6 核心悬念：「黄金轮回」预言

**预言原文（古印加克丘亚语）**
> *"当 Inti 的血脉在背叛中归来，*
> *太阳将坠落黄金之海，*
> *不是终结，而是轮回的终点，*
> *蚂蟥将从黄金中苏醒，*
> *带走所有的王。"*

**帝国的官方解读**
- "背叛"指外部敌人，"归来"指帝王血脉的永恒延续
- "太阳坠落"指王朝更替，是正常的轮回
- "蚂蟥"指外部侵略者，会被帝国清除

**真正的含义（玩家逐步发现）**
- "背叛中归来"：玩家这一脉被帝国放逐的血统，在帝国内部觉醒
- "太阳坠落黄金之海"：Inti 帝国依赖的黄金纳米技术将失控
- "轮回的终点"：循环时间观的终点是重启，不是延续
- "蚂蟥从黄金中苏醒"：**Aya-Tech 的副作用**——被黄金纳米液强化的赛博格将失去神性，只剩兽性
- "带走所有的王"：**革命**——不是暗杀帝王，而是终结整个神权阶级

### 2.1 玩家身份

**名字**：Killa Incan Valle（开发代号可简化）

**背景**：
- Inca Tech 特种作战部队最高指挥官
- 十二大家族之一"Valleys"家族血统（已被流放但家族不知情）
- 战绩：平定 3 次部落叛乱，指挥 7 次"清洗行动"
- 性格：服从、荣誉感强、有愧疚——服从是因为不知道真相

**转折点**：
- 任务中意外获得一块刻有古预言的 Qhipu 石板
- 石板与自己的血统产生神经共振——触发了 Aya-Tech 的异常反应
- 逃离时发现：自己的"叛乱"被帝国定性为叛国，而自己原本任务是**屠杀反叛者**

**叙事立场**：内部反叛者——熟悉帝国运作，但选择对抗

### 3.1 核心循环

```
观察（鹰眼扫描）

- **喀麦朗响（定时噪音）** = 延时陷阱，不可操控，预设延迟，适合预设路线触发

## 第三部分·扩展：玩法系统完整设计

---

### 3.5 核心循环（完整版）

游戏存在三个嵌套层级的循环，从微观到宏观逐一成立：

#### 3.5.1 微观循环（30s - 3min / 单区域）

```
[鹰眼扫描] → [信息处理] → [路线决策] → [工具组合] → [执行移动]
                                                           ↓
                              [感知反馈] ← [警戒状态变化] ← [敌人反应]
                                   ↓
                              [适应调整] → (新循环)
```

| 阶段 | 玩家行为 | 系统反馈 |
|---|---|---|
| **扫描** | 鹰眼扫描（CD 3s），读取敌人视野/路线/可交互物 | HUD 高亮威胁源，敌人橙色轮廓 |
| **信息处理** | 规划路线，识别敌人模式（巡逻节奏/感知盲区）| 心理地图形成 |
| **路线决策** | 选择：主路线/侧翼/管道/破坏墙 | 路线确认 |
| **工具决策** | 组合：烟雾+诱饵 / EMP+终端 / 纯潜行 | 工具消耗计划 |
| **执行移动** | 移动、工具使用、隐蔽 | 动静值产生 |
| **敌人反应** | 状态变化（正常→怀疑→搜索→警戒） | HUD 状态更新 |
| **适应调整** | 脱离/反杀/等待/改变路线 | 新一轮循环 |

**微观循环质量标准**：每一轮循环玩家都应有"我知道发生了什么，我在做什么，我知道下一步"的感受。

#### 3.5.2 中观循环（单关 / 12-22min 当前实装窗口）

```
[任务简报] → [区域 A 潜入] → [信息收集/资源获取] → [区域 B 潜入]
                                                                        ↓
                                           [终局目标] ← [核心终端破解/目标发现]
                                                  ↓
                                           [撤离] → [关卡评价] → [资源结算]
```

| 阶段 | 内容 | 反馈 |
|---|---|---|
| **任务简报** | 文字弹窗，Willa 通讯，目标标记 | 明确主/次目标 |
| **区域潜入** | 3-5 个区域依次穿越 | 警戒状态积累 |
| **信息收集** | 终端/文档/Qhipu 幻象（可选）| 叙事推进 |
| **终局目标** | 核心终端/目标击杀/数据获取 | 任务完成判定 |
| **撤离** | 返回撤离点，触发关卡结算 | 评价产生 |
| **关卡评价** | 评级（S/A/B/C）+ 资源奖励 + 工具解锁 | 反馈闭环 |

#### 3.5.3 宏观循环（5 关 / 10-20h）

```
[序章/任务 01] → [任务 02] → [任务 03] → [任务 04] → [终章 05]
      ↓                ↓              ↓              ↓            ↓
  工具解锁        工具解锁       工具解锁      工具解锁      终局抉择
      ↓                ↓              ↓              ↓
  血统共振        血统记忆       血统记忆      血统记忆
      └────────────────┴──────────────┴──────────────┘
                         叙事层层递进
```

**宏观循环质量标准**：每一关的完成都应让玩家感受到"我更强了/我知道了更多"的双重满足。

#### 3.5.4 循环闭合验证（MDA）

| 循环层级 | Mechanics（规则）| Dynamics（行为）| Aesthetics（体验）|
|---|---|---|---|
| **微观** | 工具 CD/弹药/视野锥/警戒状态 | 观察→决策→执行→反应 | 紧张感 + 掌控感 |
| **中观** | 区域结构/终端破解/撤离条件 | 资源管理/路线规划/意外应变 | 成就感 + 策略感 |
| **宏观** | 关卡解锁/工具解锁/叙事揭示 | 技能成长感/世界观理解 | 沉浸感 + 命运感 |

### 3.6 战斗系统

#### 3.6.1 设计原则

**战斗是潜行失败的应急方案，而非主动选择。**

| 原则 | 具体含义 |
|---|---|
| **应急而非主动** | 战斗在玩家被明确发现后触发，非玩家主动发起 |
| **高风险高成本** | 战斗带来高暴露风险，敌人会呼叫增援 |
| **近身为主** | 远程火力是最后手段，徒手/CQC 是主要战斗方式 |
| **快速结束** | 战斗应尽可能短，拖长则惩罚增加 |

#### 3.6.2 战斗触发条件

| 触发条件 | 系统响应 |
|---|---|
| 敌人进入「警戒」状态 5s 内未脱离 | 切换至战斗模式 |
| 玩家主动开枪（睡眠弹除外） | 立即进入「警戒」，敌人呼叫增援 |
| 敌人近距离（3m 内）直接看到玩家 | 立即进入「警戒」，无反应窗口 |
| Saqueos 主动攻击玩家 | 强制进入战斗（Saqueos 无法谈判） |

#### 3.6.3 战斗模式系统

**进入战斗模式后：**
- 玩家获得临时「战术视角」（第三人称，跟随摄像机）
- 敌人 AI 切换至「战斗行为」（不再是潜行巡逻）
- 警戒状态不再回退——必须杀出或脱离，否则任务失败

| 战斗行为 | 描述 |
|---|---|
| **锁定移动** | 敌人向玩家移动，保持视线接触 |
| **射击** | 3m 内开枪，命中率高 |
| **呼叫增援** | 30m 内友军向玩家位置移动 |
| **包抄** | 2 名以上敌人时，分队包抄玩家位置 |

#### 3.6.4 玩家战斗能力

| 能力 | 描述 | 数值 |
|---|---|---|
| **徒手格斗** | 近身 CQC，击倒敌人 | 2m 范围，0.5s 施放 |
| **背刺** | 从背后击晕，无声击杀 | 背对 2m，1s 施放 |
| **绳技击杀** | 悬挂状态向下击杀 | 悬挂点，0.3s 施放 |
| **睡眠弹射击** | 远程静音击倒 | 15m 距离，命中后入睡 20s |
| **紧急翻滚** | 快速位移，脱离战斗 | 3m 位移，无敌帧 0.2s |

#### 3.6.5 生命值系统

| 参数 | 数值 | 设计意义 |
|---|---|---|
| **玩家 HP** | 5 格（5 次 hits） | 低，逼使避免战斗，但可承受偶发失误 |
| **恢复方式** | 仅「医疗站」恢复（每关 1 处） | 战斗惩罚极高 |
| **强制医疗** | HP=0 后，自动使用急救包（非道具，1 次/关）| 软死亡机制 |
| **HP=0** | 任务失败，重新加载检查点 | 硬惩罚 |

#### 3.6.6 敌人战斗数值

| 敌人类型 | HP | 伤害 | 移动速度 | 射击精度 |
|---|---|---|---|---|
| **普通士兵** | 1 格 | 1 格/hit | 正常 | 中 |
| **强化士兵** | 2 格 | 1 格/hit | 快 | 高 |
| **重型兵** | 3 格 | 2 格/hit | 慢 | 低 |
| **Quipucamayoc** | 1 格 | 1 格/hit + 神经干扰 | 正常 | 低（近身为主）|
| **Saqueos** | 2 格 | 1 格/hit | 不规律 | 低 |

#### 3.6.7 战斗反馈

| 反馈类型 | 具体表现 |
|---|---|
| **命中反馈** | 屏幕边缘血红 + 画面抖动 |
| **击杀反馈** | 敌人消失 + 短暂无警觉窗口（其他敌人未响应）|
| **医疗站反馈** | HP 缓慢恢复 + 禁用工具 5s（强制停止）|
| **任务失败反馈** | 屏幕渐黑 + "任务失败" + 检查点重新加载 |

### 3.7 经济系统

#### 3.7.1 核心资源

| 资源名称 | 类型 | 用途 |
|---|---|---|
| **信用点（Credit）** | 货币 | 购买工具/升级 |

| **工具充能** | 冷却/能量 | 使用工具 |
| **情报碎片** | 收集品 | 解锁隐藏文档/背景故事 |

#### 3.7.2 资源循环（Machinations 模型）

```
【来源 Sources】
    │
    ├── 任务完成奖励（固定 + 评级加成）
    ├── 次要目标完成（收集品/零击杀/限时通关）
    ├── 关卡内补给箱（工具弹药/信用点）
    └── Qhipu 幻象收集（情报碎片）
            │
            ▼
    【存储 Pools】
    │
    ├── 信用点余额（跨关卡累积）
    ├── 工具栏（当前关卡可用工具配置）
    ├── 弹药计数（每关重置基础量）
    └── 充能状态（CD/能量条）
            │
            ▼
    【消耗 Drains】
    │
    ├── 购买工具（消耗信用点）
    ├── 工具使用（弹药减少/CD 触发）
    ├── 终端破解（消耗时间，间接消耗安全窗口）
    └── 墙壁破坏（消耗时间）
            │
            ▼
    【转化 Converters】
    │
    ├── 信用点 × 时间 → 新工具/弹药补给
    ├── 情报碎片 → 背景叙事解锁
    └── 任务评价（S/A/B/C） → 信用点倍率加成
```

| 规则 | 说明 |

| **关卡内无商店** | 补给全靠关卡内拾取，无额外购买机会 |

### 3.8 成长系统

#### 3.8.1 设计原则

**成长服务于叙事，而非替代叙事。**

| 成长类型 | 描述 | 与叙事关系 |
