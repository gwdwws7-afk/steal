## N1. Willa 通讯文本

Willa 是星星之子（Ayllu astral）的联络人，前 Quipucamayoc（数据祭司）。她通过加密通讯频道与玩家联系，代号"鸦"。

---

### N1.1 关卡 01：黄金雨

**触发时机：任务开始**
```
[鸦 - 加密频道]
Killa，我知道你被派去工业区镇压反叛者。

但我要你完成任务后，在目标区域找一个石板。
一个刻着古印加克丘亚语的石板。

不要问为什么。
不要交给任何人。
完成后，在旧港 7 号仓库找我。

我们等了你很久了，Valleys 的孩子。
```

**触发时机：发现石板**
```
[鸦 - 加密频道]
你找到了。

那是 Qhipu 石板，古代用来存储数据的。
你能感受到它，对吗？和你的血统共振。

你看到的是什么？
画面？声音？

无论如何，不要让帝国得到它。
如果被发现了...你就是叛逃者了。

做你该做的事，Killa。
```

**触发时机：叛逃开始**
```
[鸦 - 加密频道]
你现在是叛逃者了。

帝国把你标记为叛国罪。
你的"清洗任务"实际上是要屠杀无辜的人。
反叛者...他们才是说真话的人。

来旧港 7 号仓库。
我知道你有很多问题。

关于你的家族。
关于你的血统。
关于 Inti Imperium 对你们 Valleys 做的一切。

我都有答案。
```
