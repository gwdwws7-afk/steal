# INTI FALL — 主角重新设计
## 核心概念：安第斯女战士 + 太阳神使

---

## 角色定位调整

**原设计问题**：普通军人 + 少量印加元素，缺乏视觉冲击力

**新设计方向**：
- 安第斯高原原住民血统的外显化
- 太阳神 Inti 战士的文化意象
- 标志性的战斗姿态（Pose）
- 融入印加编织图案和金属工艺

---

## 主角外观设计（强化版）

### 体型与气质

| 元素 | 设计 |
|---|---|
| **身高** | 175cm，略高于平均女性 |
| **体型** | 肌肉线条分明，但不过分强壮——安第斯长跑者体型 |
| **肤色** | 深古铜色（安第斯高原居民特征）|
| **面部** | 高颧骨、鹰钩鼻、深邃眼窝——克丘亚人特征 |
| **眼神** | 锐利、坚定、有复仇之火 |
| **发色** | 黑色长辫，安第斯传统编发 |
| **发饰** | 银质发环，太阳神 ray 装饰 |
| **头饰** | 可拆卸额带（战斗时佩戴）——金色太阳纹样 |

### 核心 Pose 设计

**默认站立姿态（战斗前准备）**：
- 重心略低，右脚前踏
- 左手自然垂于腰侧
- 右手抬起至胸前，手掌张开（暗示"停止"或"准备战斗"）
- 头部微微侧倾，目光直视前方
- 姿态暗示：随时准备行动的战斗姿态

### 服装设计（强化印加元素）

**材质与配色**：
```
主色：深靛蓝 #1a1a3e（安第斯夜空）
金色：#d4a017（太阳神的神圣色）
铜色：#b87333（安第斯金属工艺）
银饰：#c0c0c0（高原银器工艺）
血红：#8b0000（祭祀用红色）
```

**上身设计**：
- **战斗背心**：深靛蓝嵌入式编织背心
  - 正面：克丘亚几何编织纹样（肩部向中心汇聚）
  - 编织纹样中嵌有金色细线
  - 背心边缘：铜质包边
- **编织纹理**：
  - 安第斯传统 "chakira" 编织图案
  - 肩部有悬垂的彩色绒球（识别身份用）
- **颈饰**：
  - 铜质圆盘颈环（直径 8cm）
  - 太阳神 Inti 纹样浮雕
  - 安第斯传统铃铛（行走时发出微弱声响）

**下身设计**：
- **战术长裤**：深靛蓝，侧面有金色条纹
- **编织腰带**：
  - 克丘亚传统编织工艺
  - 腰带中央：黄金圆盘扣（太阳神符号）
  - 悬挂：小型工具袋（铜扣封口）
- **小腿绑带**：
  - 安第斯传统 "shucos" 绷腿带
  - 红黑相间编织图案
  - 棉质 + 羊绒材质

**足部**：
- **战斗靴**：低帮，铜色包头
- 靴面有编织纹理
- 鞋底：防滑橡胶

| 饰品 | 位置 | 描述 | 功能暗示 |

## 姿态（Pose）设计说明

### 为什么 Pose 重要

在潜行游戏中，主角的姿态设计需要传达：
1. **角色气质**：战士的警觉与克制
2. **文化身份**：安第斯高原原住民的骄傲
3. **战斗准备**：随时可以行动的紧张感

### 三种标志性 Pose

| Pose | 场景 | 描述 |
|---|---|---|
| **战备姿态** | 默认站立 | 重心略低，右手抬起张开，凝视前方 |
| **潜行姿态** | 匍匐/掩体 | 身体压低，左手前探，右手控制工具 |
| **绳技姿态** | 悬挂移动 | 身体收紧，单手悬挂，另一手控制 |
| **击杀姿态** | 背刺 | 从背后快速压制，无声 |

### 表情设计

| 状态 | 表情 |
|---|---|
| **默认** | 锐利、坚定 |
| **被检测到** | 紧张、咬牙 |
| **击杀成功** | 冷酷、无声 |
| **血统共振** | 恍惚、眼中金光闪烁 |
| **被追击** | 恐惧与决心并存 |
