## S1. 角色移动系统

### S1.1 基础移动参数

| 参数 | 数值 | 说明 |
|---|---|---|
| 行走速度 | 4.5 m/s | 正常移动，不会触发敌人听觉 |
| 奔跑速度 | 7.0 m/s | 消耗耐力，触发敌人听觉 |
| 匍匐速度 | 1.5 m/s | 最低速度，完全隐蔽移动 |
| 绳技悬挂速度 | 3.0 m/s | 正常行走速度的 66% |
| 绳技横向移动 | 2.0 m/s | 悬挂状态下横向移动 |
| 攀爬速度 | 1.0 m/s | 墙壁/管道边缘攀爬 |
| 翻滚距离 | 3.0 m | 紧急翻滚位移 |
| 翻滚无敌帧 | 0.2 s | 战斗中紧急脱身 |
| 耐力条 | 100% | 100 单位 |
| 耐力消耗（奔跑）| 15%/秒 | 持续奔跑时消耗 |
| 耐力恢复 | 10%/秒 | 停止奔跑后恢复 |
| 耐力消耗（匍匐）| 0%/秒 | 匍匐不消耗耐力 |

### S1.2 移动状态机

```
[待机]
  │ W/A/S/D
  ▼
[行走] ──── Shift ────→ [奔跑]
  │                           │
  │ C键                       │ 松开/耐力耗尽
  ▼                           ▼
[匍匐] ←──────────────── [耐力耗尽]
  │                           │
  │ C键/靠近低矮掩体          │ 耐力恢复中
  ▼                           ▼
[掩体站立]              [恢复奔跑]
```

### S1.3 掩体系统

| 掩体类型 | 描述 | 效果 |
|---|---|---|
| **低掩体（<1m）** | 矮墙、箱子、机械设备 | 站立可挡下半身视觉，蹲下完全隐蔽 |
| **高掩体（1-2m）** | 标准墙、柱子、车辆 | 站立可挡全部视觉，需匍匐穿越 |
| **完全掩体（>2m）** | 厚墙、结构柱、装甲门 | 完全遮挡，无视任何敌人视线 |
| **破坏掩体** | 黄色标记墙体 | 可用墙壁破坏开凿通道 |
| **悬挂点** | 横梁、管道、梯子 | 允许绳技悬挂移动 |
| **通风管入口** | 墙面通风口 | 允许进入通风管系统 |

**掩体交互规则**：
- 靠近掩体 1m 范围内，按 E 进入掩体
- 掩体内按 Shift 探出身（暴露上半身，视野扩大，但移动速度 -50%）
- 松开 Shift 返回掩体
- 掩体内按 C 键匍匐穿过低矮空间

### S1.4 绳技系统

| 参数 | 数值 |
|---|---|
| 悬挂最大时长 | 无限制 |
| 悬挂击杀窗口 | 敌人进入悬挂点正下方 2m 范围 |
| 悬挂击杀施放 | 0.3s |
| 悬挂击杀音效 | 无声 |
| 悬挂状态敌人发现率 | 0%（敌人不抬头）|
| 悬挂点最小间距 | 3m |
| 悬挂点最大间距 | 8m |
| 绳技速度（升级后）| 3.9 m/s（+30%）|

**绳技使用流程**：
1. 鹰眼扫描显示所有悬挂点（高亮金色）
2. 接近悬挂点 2m 范围内，按 E 抓住悬挂物
3. WASD 移动（绳技速度）
4. 悬挂击杀：敌人进入悬挂点下方范围，按 Q 执行（0.3s 施放）
5. 松开 E 下降

### S1.5 通风管系统

| 参数 | 数值 |
|---|---|
| 进入方式 | 通风口 1m 范围内按 E |
| 移动速度 | 1.5 m/s（匍匐速度）|
| 敌人感知 | 完全不可检测 |
| 出口标记 | 鹰眼扫描高亮（绿色）|
| 管径要求 | 仅单人可进入 |

**通风管限制**：
- 管内不可使用工具
- 管内不可进行攻击
- 管内视野有限（第一人称视角切换为第三人称俯视）
- 紧急离开需找到出口（无法中途破管）

## S2. 潜行系统

### S2.1 阴影隐藏

| 条件 | 效果 |
|---|---|
| 站立于阴影中 | 敌人视觉距离 ×0.7 |
| 蹲下于阴影中 | 敌人视觉距离 ×0.5 |
| 悬挂状态 | 敌人发现率 0% |
| 阴影 + 匍匐 |敌人视觉距离 ×0.3 |

**阴影定义**：
- 照度 < 30 lux 的区域视为阴影
- 鹰眼扫描下阴影区域显示为深蓝色
- 敌人视野锥经过阴影区域时收缩（视觉削弱）

**环境光源类型**：
| 光源 | 照度 | 隐蔽效果 |
|---|---|---|
| 直射阳光 | 100,000 lux | 无阴影，无法隐蔽 |
| 霓虹灯（近）| 1,000-5,000 lux | 无阴影盲区 |
| 霓虹灯（中）| 500-1,000 lux | 小面积阴影 |
| 霓虹灯（远）| < 500 lux | 大面积阴影 |
| 紧急红光 | 50-200 lux | 良好阴影条件 |
| 完全灭灯 | 0 lux | 最佳阴影（敌人只能靠听觉）|

### S2.2 视野检测逻辑

**检测优先级**：
```
1. 玩家在敌人正后方（盲区）→ 立即背刺机会
2. 玩家在敌人视野锥内 + 阴影不足 → 进入「警戒」
3. 玩家不在视野锥但在通信范围 → 触发「怀疑」
4. 玩家不在任何感知范围 → 未察觉
```

**检测触发流程**：
```
每帧检测：
  │
  ├── 玩家在视野锥内？
  │     ├── 是 → 进入视觉检测
  │     │     ├── 阴影足够？
  │     │     │     ├── 是 → 不触发，继续
  │     │     │     └── 否 → 触发「警戒」
  │     │     └── 距离 < 3m？→ 直接触发「警戒」（无反应时间）
  │     └── 否 → 进入听觉/通信检测
  │
  ├── 玩家产生声音（行走/奔跑/工具）？
  │     ├── 是 → 进入听觉检测
  │     │     ├── 声音类型 = 行走？→ 5m 范围
  │     │     ├── 声音类型 = 奔跑？→ 12m 范围
  │     │     ├── 声音类型 = 工具？→ 8m 范围
  │     │     └── 范围内有敌人？→ 触发「怀疑」
  │     └── 否 → 进入通信检测
  │
  └── 敌人收到通信（队友警报）？
        ├── 是 → 触发「警戒」（立即，无视距离）
        └── 否 → 未察觉
```

### S2.3 警戒状态升级条件

| 从状态 | 升级条件 | 触发时间 |
|---|---|---|
| 未察觉 → 怀疑 | 听到声音 / 队友通信 / 看到动静 | 即时 |
| 怀疑 → 搜索 | 怀疑持续 3s 内未确认 | 3s |
| 搜索 → 警戒 | 亲眼看到玩家 / 睡眠弹效果 / 背刺 | 即时 |
| 警戒 → 全面警报 | 警戒状态持续 5s | 5s |
| 全面警报 → 任务失败 | 全面警报持续 30s 内未脱离 | 30s |

### S2.4 警戒状态降级条件

| 当前状态 | 降级条件 | 降级目标 |
|---|---|---|
| 怀疑 → 未察觉 | 3s 内无任何触发 | 返回原岗位 |
| 搜索 → 未察觉 | 5-10s 内未确认 | 返回原岗位 |
| 警戒 → 搜索 | 5s 内未再次看到玩家 | 进入搜索 |
| 搜索 → 未察觉 | 10s 内未确认 | 返回原岗位 |
| 全面警报 | 无法降级 | 必须脱离或死亡 |

### S2.5 悬挂击杀系统

| 参数 | 数值 |
|---|---|
| 触发条件 | 敌人进入悬挂点正下方 2m |
| 敌人朝向 | 任意方向均可 |
| 施放时间 | 0.3s |
| 击杀效果 | 敌人立即入睡（不触发警报）|
| 音效 | 无声 |
| 视觉 | 快速绳降动作，敌人倒下 |
| 数量限制 | 无（每个悬挂点可多次使用）|

**悬挂击杀流程**：
```
每帧检测：
  │
  ├── 玩家处于悬挂状态？
  │     ├── 是 → 进入敌人检测
  │     │     ├── 敌人进入悬挂点 2m 半径？
  │     │     │     ├── 是 → 显示击杀提示图标
  │     │     │     └── 否 → 无提示
  │     │     └── 按 Q 键？
  │     │           ├── 是 → 执行悬挂击杀
  │     │     │     ├── 施放动画 0.3s
  │     │     │     └── 敌人入睡 20s
  │     │     └── 否 → 继续
  │     └── 否 → 结束
```

## S4. 敌人 AI 状态机（Unity 实现）

### S4.1 状态枚举

```csharp
public enum EnemyState
{
    Idle,           // 未察觉 - 正常巡逻/站立
    Suspicious,      // 怀疑 - 暂停查看可疑方向
    Searching,       // 搜索 - 离开岗位向可疑点移动
    Alerted,        // 警戒 - 确认位置，锁定玩家
    FullAlert       // 全面警报 - 区域封锁，增援到场
}
```

### S4.2 状态机核心

```csharp
public class EnemyAIController : MonoBehaviour
{
    [Header("状态机")]
    public EnemyState currentState = EnemyState.Idle;
    private EnemyState previousState;
    
    [Header("感知参数")]
    public float visionDistance = 15f;        // 视觉距离
    public float visionAngle = 60f;           // 视觉角度
    public float hearingDistance = 5f;        // 听觉-行走
    public float runHearingDistance = 12f;    // 听觉-奔跑
    public float toolHearingDistance = 8f;   // 听觉-工具
    public float communicationRange = 30f;   // 通信范围
    
    [Header("时间参数")]
    public float suspiciousDuration = 3f;     // 怀疑持续时间
    public float searchDuration = 8f;         // 搜索持续时间
    public float alertDuration = 5f;          // 警戒持续时间
    public float fullAlertDuration = 30f;     // 全面警报持续
    
    private float stateTimer = 0f;
    private Vector3 lastKnownPlayerPosition;
    
    [Header("巡逻路径")]
    public Transform[] patrolPoints;
    private int currentPatrolIndex = 0;
    
    [Header("组件引用")]
    public Transform visionOrigin;
    public LayerMask playerLayer;
    public LayerMask obstacleLayer;
    
    void Update()
    {
        switch (currentState)
        {
            case EnemyState.Idle:
                HandleIdle();
                break;
            case EnemyState.Suspicious:
                HandleSuspicious();
                break;
            case EnemyState.Searching:
                HandleSearching();
                break;
            case EnemyState.Alerted:
                HandleAlerted();
                break;
            case EnemyState.FullAlert:
                HandleFullAlert();
                break;
        }
    }
    
    // ==================== 状态处理 ====================
    
    void HandleIdle()
    {
        // 巡逻逻辑
        Patrol();
        
        // 检测玩家
        if (CanSeePlayer())
        {
            TransitionTo(EnemyState.Alerted);
            return;
        }
        
        if (CanHearPlayer())
        {
            TransitionTo(EnemyState.Suspicious);
            return;
        }
        
        if (ReceivedRadioAlert())
        {
            TransitionTo(EnemyState.Alerted);
            return;
        }
    }
    
    void HandleSuspicious()
    {
        // 暂停，查看可疑方向
        LookAtSuspiciousDirection();
        
        stateTimer += Time.deltaTime;
        
        if (CanSeePlayer())
        {
            TransitionTo(EnemyState.Alerted);
            return;
        }
        
        if (stateTimer >= suspiciousDuration)
        {
            // 超时未确认，返回岗位
            TransitionTo(EnemyState.Searching);
        }
    }
    
    void HandleSearching()
    {
        // 向可疑点移动
        MoveToSuspiciousPoint();
        
        stateTimer += Time.deltaTime;
        
        if (CanSeePlayer())
        {
            TransitionTo(EnemyState.Alerted);
            return;
        }
        
        if (stateTimer >= searchDuration)
        {
            // 超时，返回岗位
            ReturnToPatrol();
            TransitionTo(EnemyState.Idle);
        }
    }
    
    void HandleAlerted()
    {
        // 锁定玩家位置，向其移动
        if (CanSeePlayer())
        {
            lastKnownPlayerPosition = player.position;
            ChasePlayer();
        }
        else
        {
            // 失去视线，进入搜索
            TransitionTo(EnemyState.Searching);
            return;
        }
        
        stateTimer += Time.deltaTime;
        
        if (stateTimer >= alertDuration)
        {
            TransitionTo(EnemyState.FullAlert);
        }
    }
    
    void HandleFullAlert()
    {
        // 区域封锁，增援到场
        CallReinforcements();
        
        if (CanSeePlayer())
        {
            ChasePlayer();
        }
        else
        {
            // 继续封锁区搜索
            SearchInZone();
        }
        
        stateTimer += Time.deltaTime;
        
        if (stateTimer >= fullAlertDuration)
        {
            // 30s 内未击杀玩家，任务失败
            GameManager.Instance.MissionFailed();
        }
    }
    
    // ==================== 感知检测 ====================
    
    bool CanSeePlayer()
    {
        // 视野锥检测
        Vector3 dirToPlayer = (player.position - visionOrigin.position).normalized;
        float angle = Vector3.Angle(visionOrigin.forward, dirToPlayer);
        float distance = Vector3.Distance(visionOrigin.position, player.position);
        
        if (angle > GetCurrentVisionAngle()) return false;
        if (distance > GetCurrentVisionDistance()) return false;
        
        // 障碍物检测
        if (Physics.Raycast(visionOrigin.position, dirToPlayer, distance, obstacleLayer))
            return false;
        
        // 阴影检测
        if (IsPlayerInShadow())
            return false;
        
        return true;
    }
    
    bool CanHearPlayer()
    {
        float distance = Vector3.Distance(transform.position, player.position);
        
        if (PlayerIsRunning() && distance <= runHearingDistance) return true;
        if (PlayerIsWalking() && distance <= hearingDistance) return true;
        if (PlayerUsedTool() && distance <= toolHearingDistance) return true;
        
        return false;
    }
    
    bool ReceivedRadioAlert()
    {
        // 检查是否有队友发送了警报
        return RadioManager.Instance.HasAlertInRange(transform.position, communicationRange);
    }
    
    // ==================== 状态转换 ====================
    
    void TransitionTo(EnemyState newState)
    {
        previousState = currentState;
        currentState = newState;
        stateTimer = 0f;
        
        OnStateEnter(newState);
        
        // 通知 RadioManager
        RadioManager.Instance.OnEnemyStateChange(this, newState);
    }
    
    void OnStateEnter(EnemyState state)
    {
        switch (state)
        {
            case EnemyState.Idle:
                ResumePatrol();
                break;
            case EnemyState.Suspicious:
                StopAndLook();
                break;
            case EnemyState.Searching:
                MoveToSuspiciousPoint();
                break;
            case EnemyState.Alerted:
                LockOnPlayer();
                break;
```

### S4.3 RadioManager（敌人通信管理）

```csharp
public class RadioManager : MonoBehaviour
{
    public static RadioManager Instance;
    
    private List<RadioAlert> activeAlerts = new List<RadioAlert>();
    
    struct RadioAlert
    {
        public Vector3 position;
        public float range;
        public float startTime;
        public float duration;
    }
    
    public void BroadcastAlert(Vector3 position, float range, float duration = 10f)
    {
        activeAlerts.Add(new RadioAlert
        {
            position = position,
            range = range,
            startTime = Time.time,
            duration = duration
        });
    }
    
    public bool HasAlertInRange(Vector3 position, float communicationRange)
    {
        foreach (var alert in activeAlerts)
        {
            if (Vector3.Distance(position, alert.position) <= alert.range + communicationRange)
                return true;
        }
        return false;
    }
    
    public void OnEnemyStateChange(EnemyAIController enemy, EnemyState state)
    {
        if (state == EnemyState.Alerted || state == EnemyState.FullAlert)
        {
            // 向所有友军广播
            BroadcastAlert(enemy.transform.position, enemy.communicationRange);
        }
    }
    
    void Update()
    {
        // 清理过期警报
        activeAlerts.RemoveAll(a => Time.time - a.startTime > a.duration);
    }
}
```
